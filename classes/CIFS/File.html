get(file, domain=nil, user=nil, pass=nil)

Creates a jcifs.smb.SmbFile object

file:required, has to be an smb url See jcifs.samba.org/src/docs/api/jcifs/smb/SmbFile.html
domain:optional
user:optional
pass:optional

If pass is not specified the CIFS server is accessed anonymously and user and domain is ignored

get_icon(file)

returns a simple string indicating the type of a file

file:is a SmbFile object
returns:one of dir, file, printer, share, workgroup, pipe, comm, server, unknown

It is used to select the right icon in the directory browser
