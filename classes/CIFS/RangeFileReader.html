Class CIFS::RangeFileReader
In: lib/jcifs.rb
Parent: FileReader

The RangeFileReader is a child of the FileReader and supports receiving of partial contents of a SmbFile

Methods

boundary   create_input   each   new   response  

Public Class methods

new(smbfile, range)
smbfile:an SmbFile
range:the value of an HTTP range header, e. g. ‘bytes=-5000‘

Currently only one byte-range-spec is supported, i. e. ‘bytes=5-10’ but NOT ‘bytes=5-10,-5,70-’. See RFC 2616 14.35.1 for details
